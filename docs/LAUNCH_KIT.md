# Launch Kit

Screen Translator を公開・発信するときに、そのまま叩き台として使える素材集です。

## 1. X 投稿案

### 投稿案 1: 価値直球

画面上の英語テキストを、
**範囲選択するだけで OCR → 翻訳**できる Windows アプリを作りました。

- コピペ不要
- その場で訳せる
- OpenAI + Tesseract ベース

海外 UI や英語アプリを読むときの面倒をかなり減らせます。

GitHub:
https://github.com/kzya/ScreenTranslator

### 投稿案 2: 共感寄り

英語 UI を読むたびに
「スクショ → OCR → コピペ → 翻訳」が地味に面倒だったので、
**画面上をドラッグするだけで翻訳するツール**を作りました。

Windows デスクトップアプリです。
ライブ翻訳ポップアップや履歴もあります。

GitHub:
https://github.com/kzya/ScreenTranslator

### 投稿案 3: 制作ログ寄り

最近作ったもののひとつ。
**Screen Translator**

画面上のテキストを選択すると、OCR + OpenAI でそのまま翻訳する Windows アプリです。

- Python / Tkinter
- Tesseract OCR

「コピペ翻訳の手間を減らす」ための小ツール。

## 4. デモ撮影チェックリスト

### 事前準備

- 翻訳対象の画面を決める
- 読み取りやすいサイズの英語テキストを表示しておく
- API キーや個人情報が映らないことを確認
- 不要な通知を切る
- 画面サイズとズームを固定する

### 撮影するカット

1. アプリ起動
2. 「画面を選択して翻訳」を押す
3. 対象範囲をドラッグで選択
4. 原文が表示される
5. 翻訳結果が表示される
6. 余裕があればコピーやライブ翻訳も見せる

### 見せ方のコツ

- 1本目は 20〜30 秒に収める
- 冒頭 3 秒で「何が便利か」を言う
- 「範囲選択だけで訳せる」を最優先で見せる
- 画面の動きが分かるようにマウスカーソルを入れる
